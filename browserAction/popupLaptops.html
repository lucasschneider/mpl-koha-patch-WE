Laptop Checkout

All laptop data is automatically stored by the MPL Koha Patch (including issue date, patron barcode, mouse, power supply, headphones, dvd player, laptop ID, and return date).

If you need to add a note to an issued laptop, view the laptop data and modify the corresponding record. Notes may only be modified on the same day the laptop was issued.
