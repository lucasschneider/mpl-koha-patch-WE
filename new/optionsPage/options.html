Additional Patron Messages

Replace several of LibLime's predefined patron messages with messages more often used by MPL circ

Validate Madison Addresses

Alerts library staff when an unacceptable or restricted Madison address has been entered

Autofill OPAC Login Number

Autofill a patron's OPAC username to the value of their library card

Lookup PSTAT

Automatically lookup PSTAT value for all libraries in the SCLS that sort by county subdivision or census tract number. If the PSTAT is able to be determined, a button will appear allowing staff to automatically select the geographically closest MPL location for the patron's home library

Autoselect "Digests Only"

Automatically selects the "Digests Only" option (when available) while selecting patron notification options

Due Date/Advanced Notice Toggle

Forces library staff to select either "Due Date" or "Advanced Notice" notification options, but not both

Generate Middle Initials

Automatically generates a patron's middle initial from the first name field

Update Account Type

Update JU -> AD and LUJ -> LU when saving a patron record if the patron is 18 years or older

Receipt Font

Determines the font size for the patron barcode printing function

Disable Dropbox Mode

Prevents staff from checking dropbox mode by disabling the checkbox
